# Home Assistant L10n

Home Assistant translation status dashboard.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install ha-l10n
```

### Fedora/RPM
```bash
sudo dnf install ha-l10n
```
